# MockLocation

MockLocation 是一个基于 `Android Test Provider` 的位置模拟工具

## Features

 - 点位模拟
   - 摇杆移动位置
   - 跳转到保存的位置
 - 路径模拟
   - 速度自由调节
   - 路径随机偏移
   - 环形路径循环模拟
   - 路径导入/导出
 - 自定义速度预设
 - 可隐藏的速度控制悬浮窗
 - 可调节摇杆大小
